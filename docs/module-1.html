Lab 1A

Variables

Create 2 numeric variables and assign values for each

y = 6

Calculate the sum of them

total = x + y

## [1] 16

Calculate the square root of the total

## [1] 4

Data Structures

Vector

v <- c(1,2,3,4)

## [1] 1 2 3 4
